## 准备工作：你需要哪些东西？

用 PuTTY 连接搬瓦工 VPS，事先要准备三样东西：

1. **VPS 的 IP 地址**
2. **SSH 端口号**（搬瓦工不是默认的 22，是随机生成的五位数）
3. **root 密码**

这三样信息都藏在搬瓦工的 **KiwiVM 控制面板**里。下面讲怎么找。

---

## 第一步：从 KiwiVM 获取 IP、端口和密码

登录搬瓦工账户后台，找到 **Services → My Services**，点击你的 VPS 对应的 **KiwiVM Control Panel** 按钮，进入控制面板。

进去之后，左侧信息栏一眼就能看到：

- **Public IP address**：这就是你的 VPS 公网 IP
- **SSH Port**：紧跟在 IP 下面，是一个五位数，比如 `28956`，注意不是 22

> 💡 搬瓦工故意把 SSH 端口设成随机非标准端口，是为了减少针对 22 端口的恶意扫描，安全设计，别填错就行。

**获取 root 密码**的步骤稍微多一步：点击左侧菜单里的 **Root password modification**，然后点击 **Generate and set new root password** 按钮，页面会立刻生成一个新密码并用绿框高亮显示。

⚠️ **重要提醒**：这个密码只出现这一次，刷新就没了。立刻复制保存，别手滑关掉。

至此，三样东西都有了：IP、端口、密码。

---

## 第二步：下载并打开 PuTTY

PuTTY 是一款免费、开源的 SSH 客户端，Windows 平台老牌工具，绿色免安装，下载后直接双击 `putty.exe` 就能用。

去 PuTTY 官网（putty.org）下载对应版本即可。安装包很小，几秒钟的事。

---

## 第三步：在 PuTTY 里配置并连接

打开 PuTTY，主界面很简洁，需要填的地方就这几个：

| 字段 | 填什么 |
|---|---|
| **Host Name (or IP address)** | 填你 VPS 的 IP 地址 |
| **Port** | 填 KiwiVM 里显示的 SSH 端口（比如 28956） |
| **Connection type** | 选 **SSH** |
| **Saved Sessions** | 随便起个名字，点 **Save** 保存，下次直接双击打开 |

填完点右下角的 **Open**。

第一次连接会弹出一个安全警告窗口，提示服务器指纹未知，直接点 **是(Y)** 继续即可（以后同一台服务器不会再提示）。

进入黑色终端窗口后：

1. 看到 `login as:` → 输入 `root`，回车
2. 看到 `Password:` → 粘贴你保存的密码（PuTTY 里**鼠标右键等于粘贴**，不会显示字符，这是正常的），回车

如果出现 `[root@xxx ~]#` 这样的提示符，就说明连接成功了。

> 🎉 恭喜，你已经进入搬瓦工 VPS 的世界了。

## 常见问题排查

**问题一：Connection refused / 连接被拒绝**
→ 大概率是 SSH 端口填错了。搬瓦工的端口是五位数，不是默认 22，回 KiwiVM 确认一下。

**问题二：输入密码后提示 Access Denied**
→ 大概率是密码复制时多了空格，或者大小写有问题。建议先把密码粘贴到记事本里检查一下格式，再复制进 PuTTY。

**问题三：PuTTY 卡在"waiting for server host key"**
→ 网络问题，可能服务器 IP 在本地被屏蔽。检查一下 IP 是否能 ping 通。

**问题四：密码不知道了**
→ 回 KiwiVM → Root password modification → 重置一次就行，新密码马上生效。

## PuTTY 的几个实用小技巧

用了才知道 PuTTY 有几个地方跟别的软件不一样，提前知道省很多麻烦：

- **右键 = 粘贴**：在 PuTTY 终端里，鼠标右键不是右键菜单，是直接粘贴剪贴板内容。复制命令然后单击右键，就粘进去了。
- **输密码没有字符显示**：这是 Linux 的安全机制，不是没有输入，正常打完回车就行。
- **保存 Session**：填完 IP 和端口后，在 Saved Sessions 里起个名字然后点 Save，下次打开 PuTTY 直接双击 Session 名就能连，不用每次重新填。
- **输入法切英文**：使用 PuTTY 时记得把输入法切换到英文模式，避免中文输入法干扰命令输入。
